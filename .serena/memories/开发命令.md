# 开发命令

## 测试
```bash
pytest tests/
pytest --durations=0 tests/
```

## 代码格式化
```bash
black .
black --check .
```

## 代码检查
```bash
flake8
pre-commit run --all-files
```

## 构建
```bash
python setup.py sdist bdist_wheel
```

## 文档
```bash
cd docs && make html
```

## Git操作
```bash
git status
git diff
git log
```
